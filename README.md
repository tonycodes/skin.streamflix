# StreamFlix - Netflix-Style Skin for Kodi 21 (Omega)

A clean, minimalist Kodi skin that brings the Netflix experience to your media center.

## Features

- **Netflix-style home screen** with hero spotlight and horizontal content rows
- **Dark theme** with Netflix red accent colors
- **Horizontal scrolling rows** for Recently Added, Continue Watching, and more
- **Progress indicators** on in-progress content
- **Smooth focus animations** with scale effects
- **Minimal dependencies** - only requires Kodi's built-in GUI

## Requirements

- Kodi 21 (Omega) or Kodi 20 (Nexus)

## Installation (with Auto-Updates)

**Step 1:** Download the repository: [is.gd/wbolqo](https://is.gd/wbolqo)

**Step 2:** In Kodi:
1. Go to **Settings → Add-ons → Install from zip file**
2. Navigate to the downloaded `repository.streamflix.zip` and install
3. Go to **Settings → Add-ons → Install from repository**
4. Select **StreamFlix Repository**
5. Go to **Look & feel → Skins**
6. Select **StreamFlix** and install

Now the skin will automatically update when new versions are released!

## Quick Install (Manual)

**Download:** [is.gd/93dtrd](https://is.gd/93dtrd)

1. Download the zip from the link above
2. In Kodi: **Settings → Add-ons → Install from zip file**
3. Navigate to the downloaded zip and install
4. Go to **Settings → Interface → Skin** and select "StreamFlix"

*Note: Manual installs won't receive automatic updates.*

## Home Screen Rows

The home screen displays the following content rows:

- **Continue Watching** - In-progress episodes
- **Recently Added Movies** - Latest movies added to your library
- **Recently Added TV Shows** - Latest TV shows added to your library
- **Popular on StreamFlix** - Most played content
- **Top Rated** - Highest rated movies

## Customization

Color definitions are in `resources/colors/defaults.xml`:
- `netflix_red` - Primary accent color (#E50914)
- `netflix_black` - Main background (#141414)
- `text_white` - Primary text color
- `text_gray` - Secondary text color
